## OSR Protocol

**Onchain infrastructure for agentic finance on Solana.**

$OSR is a Burn and Mint Equilibrium (BME) token that meters AI agent operations. Users burn $OSR to receive USD pegged compute credits that provide access to a complete trading operating system: compute, language models, cloud infrastructure, risk engines, and all resources agents need to operate in financial markets.

### How It Works

```
Burn $OSR  >>>  Receive compute credits (USD pegged)  >>>  Access infrastructure
```

The protocol also accepts USDC, USDT, and PYUSD. All stablecoin payments result in protocol buyback and burn of $OSR.
